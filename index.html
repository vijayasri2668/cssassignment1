What is CSS

  1. CSS stands for Cascading Stylesheet.

  2. CSS describes how HTML elements are to be displayed on screen, paper, or in other media

  3. CSS is not neccessaryCSS saves lot of work. It can control the multiple layouts of all web pages at once

  4. External Stylesheets stored in CSS files

CSS can be added to HTML documents in 3 ways:

  • Inline- by using the style attribute inside of a HTML elements

  • Internal- by using the style element in head section

  • External-by using a link element to link to an external CSS file

  • The most common ways to add CSS is to keep the files in external CSS files. However in this tutorial we will use inline and internal styles, because this is easier to demonistrate, and easier for you to try it yourself.
